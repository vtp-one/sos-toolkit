# SOS-Service
A non-complete list of available services.

Services are not described in any detail at this time.

Not all services are fully functional.

## chromadb
Use docker and ollama service to provide a consistent ChromaDB server


## dev_mode
Enable special actions that make debugging development of an sos-system.yaml file easier


## local_cert
Automatically load a local ssl cert using a docker image for local development work


## local_data
Manage a special directory in a system's internal directory for storing local data

## mongodb
Use docker to provide a consistent MongoDB server

## ollama
Use docker to provide a consistent ollama server


## test_service
A service for testing service functionality
